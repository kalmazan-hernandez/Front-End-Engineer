CSS Selectors

Name Example Description
Type Selectors a {color: green;} The type selector selects all the attribute elements and change them green.
Universal Selectors * { font-family: "Times New Roman"; } The universal selectors has an asterisk to select all elements in the brackets and change the font to Times New Roman.
Class Selectors .title { color: Red; } The class selectors selects the class title and change it to Red.
ID Selectors #article-title { font-family: cursive; } The ID selectors has a hastag symbol to select the ID and change the font-family to cursive.
Attribute Selectors a[target] { color: Orange;} The attribute selectors targets elements with a specific attribute such as a word and changes the color Orange.
Pseudo-class selectors a:hover { color:green;} The pseudo-class selectors are changes that affect the users actions such as hovering over a link that changes color.
Chaining selectors h2.destination { font-family: Tahoma;} The chaining selectors are multiple css selectors with specific tag to be used on.
Multiple selectors h1, th { font-family: Calibri;} The multiple selectors are used to not repeat rules that can be used only once.
